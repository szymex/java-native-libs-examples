JAVA to native integrations
===========================

This repository contains examples for usage of native code from java, with several libraries.
For simplicity no build tool (like maven) is used.

## Requirements

- make
- jdk 8
- g++
- JAVA_HOME environment

### Build and run

        make all
        make run
